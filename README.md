## 基于区块链的拍卖交易系统

#### 部署节点

```shell
gradlew deployNodes
cd deploy/nodes
sh runnodes
```

#### 启动后端

```shell
gradlew bootJar
java -jar web/build/libs/auction-app-*.jar
```
